# Budget Whisperer

Budget Whisperer is a beginner-friendly personal finance analysis project that helps users understand spending habits by organizing store names, categorizing expenses, and generating spending summaries.

## Features

- Reads store and receipt data from text files
- Categorizes spending into groups such as grocery, food, home, and other
- Uses dictionaries to organize store names by category
- Analyzes spending patterns using Python and Jupyter Notebook
- Generates simple reports to help users understand where their money is going
